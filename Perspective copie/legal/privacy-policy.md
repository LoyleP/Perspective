# Politique de confidentialité

**Dernière mise à jour :** 15 avril 2026

## Introduction

Perspective ("l'Application") respecte votre vie privée. Cette politique explique comment nous traitons vos données.

## Données collectées

L'Application stocke localement sur votre appareil :
- **Préférences de notifications** : Votre choix d'activer ou désactiver les notifications push
- **Préférence de thème** : Mode clair ou sombre
- **État d'onboarding** : Si vous avez complété l'introduction
- **Signets** : Articles que vous avez sauvegardés localement

**Aucune donnée personnelle identifiable n'est collectée.**

## Stockage des données

Toutes les données sont stockées localement dans UserDefaults sur votre appareil. Aucune donnée n'est transmise à nos serveurs.

## Services tiers

L'Application accède en lecture seule à une base de données Supabase pour récupérer :
- Articles de presse français
- Métadonnées des sources médiatiques
- Classifications de spectre politique

Cet accès est anonyme. Aucune donnée utilisateur n'est envoyée à Supabase.

## Notifications push

Les notifications sont optionnelles. Vous pouvez les désactiver à tout moment dans :
- Réglages de l'Application → Alertes
- Réglages iOS → Notifications → Perspective

Nous n'utilisons pas les notifications pour du suivi publicitaire.

## Aucun suivi

L'Application ne contient :
- Aucun SDK de publicité
- Aucun identifiant publicitaire (IDFA)
- Aucun suivi inter-applications
- Aucune analytique tierce

## Navigation web

Lorsque vous ouvrez un article, l'Application utilise Safari intégré (WKWebView) pour afficher le site web du média. Les sites web visités peuvent collecter leurs propres données selon leurs politiques de confidentialité respectives.

## Suppression des données

Pour supprimer vos données :
1. Désinstallez l'Application
2. Toutes les données locales (signets, préférences) seront supprimées automatiquement

## Droits des utilisateurs

Conformément au RGPD (Règlement Général sur la Protection des Données) :
- Vos données restent sur votre appareil
- Aucun compte utilisateur n'est requis
- Aucune donnée n'est vendue ou partagée avec des tiers
- Vous contrôlez totalement vos données via la désinstallation
